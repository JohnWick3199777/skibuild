# ski-docs

## Purpose

Generate standardized documentation artifacts for each skill lifecycle run.

## Responsibilities

- Produce `SKILL.md` module docs.
- Generate or update repository `README.md`.
- Emit `CLAUDE.md` for agent guidance.
- Normalize metadata for export.

## Inputs

- Verified outputs from research/flow/dev/test stages.
- Project metadata and naming conventions.
- Documentation templates.

## Outputs

- Consistent markdown artifacts.
- Structured metadata bundle for export.
- Documentation change log.

## Quality checks

- Required sections are present in each artifact.
- Terminology is consistent across files.
- Export metadata matches tested implementation state.
